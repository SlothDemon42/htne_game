RUN GAME

	Run last cell manually, allow download

GAME CONTROLS
	1) Place goal with mouse
	2) Place enemies with mouse
	3) Place player with mouse
	MOVEMENT:
		a:left
		d:right
		SPACE:jump
		t=toggle platform view
		r=reset
